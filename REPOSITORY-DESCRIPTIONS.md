## Repository Organization Tips

### Naming Conventions
- **Research Projects:** `project-name-year` (e.g., `cassava-disease-detection-2024`)
- **Code Tools:** `tool-name` (e.g., `agricultural-ai-toolkit`)
- **Datasets:** `dataset-name-data` (e.g., `zanzibar-cloves-data`)
- **Publications:** `paper-title-code` (e.g., `placental-imaging-ai-code`)

### Repository Structure
```
repository-name/
├── README.md
├── LICENSE
├── requirements.txt
├── src/
├── data/
├── docs/
├── examples/
└── tests/
```
